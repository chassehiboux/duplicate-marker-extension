## Задача

1. Добавить отдельный конфиг-файл в `PyramidNewYear` для быстрого включения/выключения показа переключателя «Новый год».
2. Добавить новую тему «Весна!» с отдельным переключателем и продуманным оформлением.
3. Сохранить совместимость с текущей логикой скриншотов (автоскрытие элементов расширения).

## Предлагаемая архитектура (после вашего согласования)

1. Новый файл конфига: `PyramidNewYear/pyramid_theme_config.js`.
2. В конфиге держим флаги:
   - `showNewYearToggle`
   - `showSpringToggle`
   - `springDefaultVariant`
   - `allowThemeOverlap` (по умолчанию `false`, чтобы одновременно активна была только одна сезонная тема).
3. Подключение в `manifest.json`:
   - сначала `PyramidNewYear/pyramid_theme_config.js`
   - затем `PyramidNewYear/pyramid_christmas.js`
   - затем новый `PyramidNewYear/pyramid_spring.js` (или расширение существующего скрипта, я рекомендую отдельный файл).
4. Для «Весна!» вводим собственные классы:
   - `spring-header-item`, `spring-label`, `material-switch-spring`
   - `spring-element`, `spring-petal-layer`, `spring-active`
5. Для режима скриншота расширяем `content.js`:
   - скрытие `.spring-*` элементов по аналогии с `.ny-*`
   - временное отключение `body.spring-active` на время снимка, затем восстановление состояния.

## Куда ставим переключатель «Весна!»

Текущее предложение:
1. Оставить «Новый год» на существующем месте (после первого крупного блока меню).
2. «Весна!» вставлять сразу рядом, как следующий элемент, чтобы оба сезонных переключателя были в одном кластере.
3. Вариант по UX: если `showNewYearToggle=false`, «Весна!» занимает его место без сдвига остального хедера.

## Визуальная концепция «Весна!»

Ниже три варианта (черновые CSS лежат в этой папке):
1. `spring-theme-option-a.css` — «Сакура»:
   - мягкий розово-зеленый градиент в заголовках;
   - падающие лепестки (легкая анимация, ограниченная плотность).
2. `spring-theme-option-b.css` — «Свежая зелень»:
   - светло-зеленые акценты без сильных эффектов;
   - плавное «дыхание» шапок и выделений.
3. `spring-theme-option-c.css` — «Минималистичная весна»:
   - деловой, почти нейтральный стиль;
   - минимум анимаций, фокус на читаемости.
