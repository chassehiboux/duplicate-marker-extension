# Интернет-референсы для темы «Весна!»

Дата подбора: 06.03.2026

## Анимации и производительность

1. MDN `@keyframes` (база по CSS-анимациям):  
   https://developer.mozilla.org/en-US/docs/Web/CSS/%40keyframes
2. MDN `prefers-reduced-motion` (доступность, снижение анимации):  
   https://developer.mozilla.org/en-US/docs/Web/CSS/%40media/prefers-reduced-motion
3. CSS-Tricks, анимированные градиенты:  
   https://css-tricks.com/animating-a-css-gradient-border/

## Готовые идеи/библиотеки

1. `sakura` (MIT), эффект падающих лепестков:  
   https://github.com/jhammann/sakura
2. Hero Patterns (CC BY 4.0), SVG-паттерны для фонов:  
   https://heropatterns.com/
3. SVG Backgrounds, набор генераторов фонов:  
   https://www.svgbackgrounds.com/

## Как использовать в проекте безопасно

1. Не копировать большие куски чужого кода напрямую.
2. Брать только идею и реализовывать своей CSS/JS-логикой.
3. Для прод-версии избегать тяжелых библиотек, если можно сделать нативно.
