# Локальные правила для Codex

1. Все файлы с кириллицей сохранять в UTF-8.
2. Русский пользовательский текст всегда оставлять на русском языке, не переводить на английский.
3. Не заменять русский текст на `?????` и не использовать `\uXXXX`-экранирование вместо обычной кириллицы в строках интерфейса.
4. После правок, затрагивающих русский текст, проверять файл на отсутствие искажений кодировки.
