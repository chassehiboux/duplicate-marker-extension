# Классификация заявок через расширение

## Спецификация

Цель: заменить медленную HtmlService-модалку `problemPicker.html` локальной модалкой расширения на странице Google Sheets.

Таблица: `1A7bsMpcMegLvTIpdRBa06TD5RxGi64Cw3goYydbujNo`.

Код расширения вынесен в отдельный блок:
- `GoogleSheets/problem_picker_config.js`
- `GoogleSheets/problem_picker_content.js`
- `GoogleSheets/problem_picker_background.js`

Модуль подключается отдельной записью `content_scripts` только для URL:

`https://docs.google.com/spreadsheets/d/1A7bsMpcMegLvTIpdRBa06TD5RxGi64Cw3goYydbujNo/*`

Листы для классификации:
- `Заявки`
- `Заявки (наши)`

Колонки определяются по заголовкам:
- `Текст заявки`
- `Проблемы`

Очередь формируется в расширении по CSV-экспорту активного листа: берутся строки, где `Текст заявки` заполнен, а `Проблемы` пустая. На реальной таблице проверено: экспорт листа `Заявки` возвращает 2545 строк, `Проблемы` находится в колонке B, `Текст заявки` в колонке T.

## Ограничение записи

Запись через UI Google Sheets не используется. На реальной таблице включён фильтр: при переходе к `B2021` интерфейс выбрал `B2029`, поэтому автоматическая вставка через DOM может записать значение в неверную строку.

Для сохранения используется лёгкий Apps Script bridge:
- расширение показывает модалку, хранит очередь и быстро переключает заявки;
- bridge принимает `sheetName`, `row`, `value`;
- bridge вызывает существующую серверную логику `finalizeProblemSelection`, чтобы сохранить категорию и выполнить очистку текста заявки.

## План

1. Добавить отдельную плавающую кнопку запуска на странице указанной таблицы.
2. Загружать CSV активного листа через `fetch(..., { credentials: 'omit' })`.
3. Показывать локальную модалку классификации с теми же вариантами проблем.
4. Сохранять выбор через bridge URL, заданный кнопкой `URL` на странице таблицы.
5. Скрывать элементы модалки при создании скриншота.

## Быстрый старт

1. Добавить файл `ProblemPickerBridge.gs` в Apps Script проект таблицы.
2. Задеплоить Apps Script как Web app.
3. Открыть таблицу и нажать плавающую кнопку `URL`.
4. Вставить URL Web app вида `https://script.google.com/macros/s/.../exec`.
5. Нажать `Классификация`.
